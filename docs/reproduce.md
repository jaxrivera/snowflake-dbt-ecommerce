# Project Reproduction Guide

### 3. dbt Configuration
Update your local `~/.dbt/profiles.yml` using the following template (do not commit this file to GitHub):
- **Account:** `<YOUR_SNOWFLAKE_ACCOUNT_LOCATOR>`
- **User:** `<YOUR_USERNAME>`
- **Password:** `<YOUR_PASSWORD>`
- **Role:** `PROJ_DE_ROLE`
- **Database:** `PROJ_ANALYTICS`
- **Schema:** `MARKETING_TRANSFORMED`
